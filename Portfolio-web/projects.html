Featured Projects

I have worked on many projects over the course of being a full stack developer as well as a Python developer. Here are a few of my live, real-world projects.

E-commerce Website

I developed a dynamic e-commerce website utilizing HTML, CSS, Tailwind CSS, JavaScript, and Flask. The project focuses on delivering a seamless and responsive user experience, combining modern design aesthetics with robust backend functionality. Tailwind CSS was employed to streamline the styling process, while Flask powers the server-side logic, ensuring efficient data handling and smooth interactions across the platform.

Personal Assistant

I created a voice-activated assistant using Python, integrating libraries such as Pyttsx3 for text-to-speech, SpeechRecognition for capturing and processing voice commands, and webbrowser for executing tasks like opening web pages. The assistant, named 'TARS,' greets the user based on the time of day and listens for commands through the microphone. It can perform various tasks, including opening project directories, telling jokes, and providing the current time. The assistant showcases a practical implementation of natural language processing and interaction with external resources, making it a useful tool for automating simple tasks.

Weather App

I developed a simple weather application using Python and the Tkinter library for the graphical user interface (GUI). The app allows users to enter a city name, fetches real-time weather data from the OpenWeatherMap API, and displays key information such as weather description, temperature, humidity, and wind speed. The app handles different scenarios, including cases where the city is not found, and updates the GUI accordingly. This project demonstrates the integration of API data retrieval with a user-friendly interface, providing a practical tool for real-time weather updates.

Web Scraping

I have extensive experience in web scraping, having successfully extracted data from various websites such as Foodpanda, Uber Eats, Instagram, and more using Python. By leveraging libraries like BeautifulSoup and requests, I developed scripts to automate the collection of valuable information, including product details, user reviews, and social media data. My web scraping projects demonstrate my ability to navigate different web structures, handle dynamic content, and efficiently process large datasets. This skillset has enabled me to gather and analyze data for various purposes, from market research to sentiment analysis.

Virtual Wifi Network

I developed a virtual WiFi network using Mininet-WiFi within a VMware environment running Ubuntu. This project involved designing and configuring a network topology with multiple access points and stations. I implemented mobility for the stations to emulate real-world scenarios, ensuring continuous WiFi connectivity across different locations. The setup and management of this network required a deep understanding of network protocols, wireless communication, and virtualization technologies.
