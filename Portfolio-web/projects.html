RAG-Based AI Chatbot

RAG-Based AI Chatbot is an advanced conversational AI system utilizing Retrieval-Augmented Generation (RAG) to deliver precise and context-aware responses. Built with LangChain, FAISS, and Google Gemini AI, it processes text documents, generates embeddings, and retrieves relevant information to answer user queries. The chatbot maintains conversational memory using ConversationalRetrievalChain, ensuring seamless interactions. Ideal for knowledge retrieval and interactive AI applications.

whatsapp automation tool

WhatsApp Automation Tool is a Python-based script designed to streamline messaging on WhatsApp. It allows users to send individual messages, bulk messages via a CSV file, and group messages efficiently. Built using the PyWhatKit library, it automates scheduled messaging with ease. The tool supports personalized bulk messaging, ensuring seamless communication while avoiding rate-limiting with controlled message intervals. Whether you're reaching out to clients, managing group announcements, or automating daily reminders, this tool simplifies the process, making WhatsApp messaging more efficient and hassle-free.

Discord bot

Ideal for server management and engagement, this bot simplifies everyday Discord interactions with efficiency.
